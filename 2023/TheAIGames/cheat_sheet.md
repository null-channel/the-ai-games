# How to run the projects:

## Java:
```bash
./gradlew run
```

## Python:
```bash
python main.py
```

## C#:
```bash
dotnet run
```
Note: you might want to put the following in the Progam.cs file:
```csharp
namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}
```

## Javascript:
```bash
npm dev
```

## Rust:
```bash
cargo run
```

## Swift:
```bash
swift run
```

## Go:
```bash
go run main.go
```
